# flight_booking / cards

P7 阶段会写入 8 个 A2UI 卡片:
- `OD_INPUT.card.yaml`        — 出发/到达/日期
- `FLIGHT_LIST.card.yaml`     — 航班列表
- `CABIN_LIST.card.yaml`      — 舱位选择
- `PASSENGER_FORM.card.yaml`  — 乘机人表单
- `OAT_BINDING.card.yaml`     — OAT 成本中心
- `PRICE_VERIFY.card.yaml`    — 价格复核
- `POLICY_DECISION.card.yaml` — 差标决策
- `ORDER_CONFIRM.card.yaml`   — 订单确认
- `ORDER_SUCCESS.card.yaml`   — 下单成功
- `CANNOT_ORDER.card.yaml`    — 不可下单
